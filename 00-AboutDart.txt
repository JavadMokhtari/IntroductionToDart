Dart is a client-optimized language for developing fast apps on any platform.
Its goal is to offer the most productive programming language for multi-platform development, paired with a flexible execution runtime platform for app frameworks. 
Dart is developed by Google with the aim of allowing developers to use an object-oriented language with static type analysis.
Dart is an open-source programming language with free-to-use advanced features that have made it a hit with software developers. 
It is an object-oriented programming language and supports all oops concepts such as classes, inheritance, interfaces and optional typing features. 
It also supports advance concepts like mixin, abstract classes, reified generic, and robust type system. 
Dart can compile to either native code or JavaScript. 
It is optimized for UI development and has a programming language specialized around the needs of user interface creation. 
It also supports productive development by allowing developers to make changes iteratively using hot reload to see the result instantly in their running app. 
Dart is fast on all platforms and can compile to ARM & x64 machine code for mobile, desktop, and backend.
